# Sama Sama

Sama Sama will serve as a centralized platform for all news and updates within UCI Kababayan's Alyansa. While Kaba offers a wide range of activities and programs, fragmented social media updates result in a level of difficulty for some students. Instead of having to venture to specific social media profiles or coming across these opportunities by chance, students can use Sama Sama to discover events, initiatives, and community resources in one unified, purpose-built platform.

## 🚀 Tech Stack Architecture

* **Frontend:** React (Vite)
* **Backend:** Node.js (Express)
* **Database & Auth:** Supabase (PostgreSQL)
* **Styling:** CSS / Tailwind

## 📂 Project Structure
* `/client`: The React frontend application (Vite). Handles the user interface, feed, and direct public data fetching.
* `/server`: The Node.js/Express backend. Handles secure logic, admin verification, and complex API routing.

## 🛠️ Setup & Installation

1.  **Clone the repository**
    git clone link
2.  **Install dependencies** (Root, Client, and Server)
    cd samasama && npm install
    cd client && npm install
    cd ../server && npm install
    
3.  **Run Development Environment** 
    cd ..
    npm run dev

The app will be running locally at http local host link
